# Editor de texto y Comandos

## Cuatro caracteristicas importantes que buscar en un Editor de Texto 
   1.Autocomplementar el código: Hace que sea mas sencillo cuando se necesita escribir grandes cantidades de código 
   2.Resaltado de Sintaxis: Es una función que toma el texto que escribe y lo hace más visible al colorearlo lo que facilidad la busquedad de errores
   3. Variedad de temas: Sobretodo para evitar la fatiga visual depsues de estar gran cantidad de horas frente al computador
   4.Amplia seleccion de extensiones disponibles: Garantiza el poder tener acceso a funciones que se necesiten en un momento determinado.
   
## Para que funcionan los siguientes comando 
   * pwd: Imprime en la consola la ruta completa del trabajo actual.
   * ls:  Muestra el contenido del directorio actual.
   * cd:  Cambia el directorio actual a la ubicación especificada.
   * mkdir: Crea un nueva carpeta con el nombre especificado.
   * touch: Crea un nuevo archivo vacío con el nombre especificado.
   
## Si ingresas estos comandos y argumentos en la línea de comandos; ¿Qué pasaría?
   * **cd projects :** Cambia el directorio actual al directorio llamado '"projects"'.
   * **mkdir new-project :** Crea un nuevo directorio dentro del directorio actual llamado '"new-project"'.
   * **touch new-project/newfile.md :** Crea un nuevo archivo llamado "newfile.md" dentro del directorio '"new-project"'. No proporciona contenido, el archivo esta vacío.
   * **cd .. :** Cambia el directorio actual al directorio padre (un nivel hacia arriba).
   * **ls projects/new-project :** Lista el contenido del directorio '"new-project"' dentro de '"projects"', mostrando el archivo recién creado '"newfile.md"'.
