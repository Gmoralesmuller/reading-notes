#HTML

## ¿Qué es HTML y por qué lo usamos?
Es un lenguaje de marcado de hipertexto que es usado para estructurar una página web. 

## ¿Cuáles son las 3 partes principales de un elemento HTML?
  * Etiqueta de Apertura
  * Contenido del elemento
  * Etiqueta de cierre

## ¿Cómo se llama cuando le das información extra a un elemento?
Generalmente se les denomina atributos

## ¿Qué es un elemento semántico?
Un elemento semántico en HTML es aquel que proporciona significado y estructura al contenido de una página web, además de ayudar a los navegadores y motores de búsqueda a comprender el propósito y la relación de las diferentes partes del contenido.
