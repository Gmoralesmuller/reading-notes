
# Javascript 

## 1. ¿Qué son las variables en JavaScript?

Una variable es un contenedor para almacenar datos. Estos datos pueden ser números, cadenas de texto, valores booleanos, objetos, funciones, etc. Las variables en JavaScript se pueden utilizar para almacenar y manipular valores dentro de un programa.

## 2. ¿Qué significa “declarar” una variable?

Declarar una variable en JavaScript significa reservar un espacio en la memoria para almacenar datos y darle un nombre único.

## 3. ¿Qué es un operador de “asignación” y qué hace?

Es el signo igual (=). Se utiliza para asignar un valor a una variable.

## 4. ¿Cómo se llama la información recibida del usuario?

 La información recibida del usuario en JavaScript generalmente se almacena en variables. Esta información puede ser ingresada por el usuario a través de formularios en una página web, eventos de clic, entrada de teclado, etc. Para capturar la información del usuario, se utilizan métodos como prompt() para obtener datos directamente del usuario a través de una ventana emergente, o se puede interactuar con elementos de formulario HTML y capturar su valor utilizando JavaScript.
