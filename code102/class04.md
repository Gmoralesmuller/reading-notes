# HTML

## 1. ¿Qué es HTML y por qué lo usamos?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar utilizado para crear y diseñar páginas web. Se utiliza para estructurar el contenido de una página web, definiendo la jerarquía y el significado de los diferentes elementos que la componen.

## 2. ¿Cuáles son las 3 partes principales de un elemento HTML?

Las tres partes principales de un elemento HTML son:

   1. Etiqueta de apertura: Es el inicio del elemento y se define mediante el nombre del elemento encerrado entre corchetes angulares (< y >). Por ejemplo, <p> indica el inicio de un párrafo.

   2. Contenido: Es el propio contenido del elemento, como texto, imágenes, otros elementos HTML, etc.

   3. Etiqueta de cierre: Es el final del elemento y se define mediante el nombre del elemento precedido por una barra inclinada (/) dentro de corchetes angulares. Por ejemplo, </p> indica el final de un párrafo.

## 3. ¿Cómo se llama cuando le das información extra a un elemento?

Cuando le das información extra a un elemento HTML, estás agregando atributos. Los atributos proporcionan información adicional sobre un elemento y se especifican dentro de la etiqueta de apertura del elemento. Por ejemplo, en el siguiente código \* <img src="imagen.jpg" alt="Descripción de la imagen"> \* , **src** y **alt** son atributos del elemento \*<img>\* , donde "src" especifica la ubicación de la imagen y "alt" proporciona un texto alternativo para la imagen.

## 4. ¿Qué es un elemento semántico?

Los elementos semánticos en HTML son esenciales para proporcionar información sobre el significado y la estructura del contenido de una página web. En lugar de simplemente especificar la apariencia visual, estos elementos ayudan a los motores de búsqueda, lectores de pantalla y otros dispositivos a comprender mejor el contenido y su contexto.

Algunos ejemplos de elementos semánticos en HTML incluyen:

- `<header>`: Define la cabecera de una página o sección.
- `<footer>`: Define el pie de página de una página o sección.
- `<nav>`: Define la navegación de una página.
- `<article>`: Define un artículo independiente y autocontenido en una página.
- `<section>`: Define una sección genérica en una página.
- `<aside>`: Define contenido relacionado pero independiente del contenido principal.

Utilizar estos elementos de manera adecuada no solo mejora la accesibilidad y usabilidad de un sitio web, sino que también ayuda en la optimización para motores de búsqueda.
