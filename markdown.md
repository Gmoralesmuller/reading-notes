# Markdown
## 1.¿Qué es?

Es un lenguaje que tiene la finalidad de permitir crear contenido de una manera sencilla de escribir, y que en todo momento mantenga un diseño legible, así que para simplificar puedes considerar Markdown como un método de escritura.

## 2.¿Por qué utilizamos Markwown?

Por la simplicidad de su sintaxis ya que se puede utilizar siempre que se necesite escribir y dar formato rápidamente, sobre todo si se quiere hacer desde dispositivos móviles.

## 3.¿Qué símbolo creará un encabezado en Markdown?

El simbolo para crear encabezados es #

## 4.Al poner el texto en negrita o cursiva para darle énfasis, ¿qué símbolo es la mejor práctica?

El simbolo de mejor práctica es *

## 5.¿Cómo se crea un vínculo con Markdown?

Puede crear un vínculo en línea escribiendo su texto entre corchetes [ ] y escribiendo la URL entre paréntesis ().

## 6.¿Cuáles son los tres símbolos que puedes usar para crear una lista desordenada (con viñetas) con Markdown?

Se puede crear una lista sin ordenar, colocando -, * o + antes de una o más líneas de texto.
